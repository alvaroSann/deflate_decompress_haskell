# DEFLATE Decompressor (RFC 1951) на Haskell

Учебная реализация распаковщика DEFLATE для курса **«Теория реализации языков программирования»** (МФТИ, 2026).  

Проект представляет собой библиотеку на языке Haskell, которая принимает на вход сжатые данные в чистом формате DEFLATE и возвращает исходную байтовую строку.

---

## Требования

* [GHCup](https://www.haskell.org/ghcup/) (используйте его для установки GHC и Cabal)
* **GHC** >= 9.0
* **Cabal** >= 3.0

---

## Быстрый старт

### 1. Клонирование репозитория
```bash
git clone [https://github.com/alvaroSann/deflate-decompress.git](https://github.com/alvaroSann/deflate-decompress.git)
cd deflate-decompress

```

### 2. Сборка проекта

```bash
cabal build

```

### 3. Интерактивное тестирование (REPL)

Запустите интерактивную среду разработки:

```bash
cabal repl

```

Внутри REPL импортируйте модули и проверьте базовые сценарии распаковки:

```haskell
import Codec.Deflate
import qualified Data.ByteString as BS

-- Тест 1: Пустой блок
let emptyDeflate = BS.pack [0x01, 0x00, 0x00, 0xff, 0xff]
decompress emptyDeflate
-- Результат: Right ""

-- Тест 2: Uncompressed (stored) блок со строкой "Hello"
let helloStored = BS.pack [0x01, 0x05, 0x00, 0xFA, 0xFF, 0x48, 0x65, 0x6c, 0x6c, 0x6f]
decompress helloStored
-- Результат: Right "Hello"

```

### 4. Запуск автоматических тестов

```bash
cabal test

```

## Проверка на произвольных данных

Вы можете сжать любую строку встроенными средствами Python, получить её DEFLATE-представление в формате HEX и убедиться, что Haskell-декодер корректно восстанавливает оригинал.

### Шаг 1. Генерация сжатого потока (HEX)

* **Вариант А: Локально (если установлен Python)**
```bash
python utils/generate_test.py "Hello, World!"

```


*Вывод программы:* `f348cdc9c9d75108cf2fca4951640000`
* **Вариант Б: Онлайн (если Python локально не установлен)**
Откройте любой онлайн-интерпретатор Python и выполните следующий скрипт:
```python
import zlib
text = "Ваш текст"
# Сжатие без заголовков zlib (wbits=-15)
print(zlib.compress(text.encode('utf-8'), wbits=-15).hex())

```


Скопируйте полученную HEX-строку.

### Шаг 2. Распаковка HEX-строки в текст

Выполните команду в корневом каталоге проекта, передав HEX-строку в качестве аргумента:

```bash
cabal run deflate-test -- f348cdc9c9d75108cf2fca4951640000

```

В консоли должен отобразиться ваш исходный текст.

## Структура проекта

Проект имеет модульную структуру, где зоны ответственности четко разделены:

| Путь к файлу | Описание модуля |
| --- | --- |
| **`src/Data/BitStream.hs`** | Побитовое чтение потока с обработкой порядка битов **LSB-first**. |
| **`src/Data/Huffman.hs`** | Построение канонических деревьев Хаффмана, чтение длин кодов и декодирование символов. |
| **`src/Codec/Deflate/GlobalTypes.hs`** | Статические константы спецификации (таблицы длин/расстояний) и фиксированные деревья. |
| **`src/Codec/Deflate/Decoder.hs`** | Разбор структуры блоков DEFLATE (`Stored`, `Fixed`, `Dynamic`) и обратная трансформация **LZ77**. |
| **`src/Codec/Deflate.hs`** | Главная точка входа библиотеки — публичный интерфейс и функция `decompress`. |
| **`app/Main.hs`** | Исполняемая консольная утилита `deflate-test` для запуска из командной строки. |
| **`utils/generate_test.py`** | Вспомогательный скрипт на Python для быстрой генерации тест-кейсов. |

## Важные примечания

* **Чистый формат:** Библиотека работает исключительно с «сырым» потоком **DEFLATE** (RFC 1951). Архивы `.gz` (gzip) или контейнеры `.zlib` содержат дополнительные мета-заголовки и контрольные суммы, которые необходимо снять перед передачей данных в декодер.
* **Строгая валидация:** Проверка целостности несжатых (`stored`) блоков выполнена в строгом соответствии со стандартом — проверяется равенство побитового дополнения размеров (`NLEN == ~LEN`).
* **Документация:** Исходный код проекта снабжен подробными комментариями на русском языке, объясняющими шаги алгоритма.
